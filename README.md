
# تطبيق المهام (أيفون و أندرويد)

## الوصف:

- تطبيق تم إنشاؤه يعمل بنظام الأندرويد والأيفون.

# Todo App (iOS & Android)

## Description:

- A Flutter project.
- It is created for iOS & android.
- The application allows the user to add the tasks that he/she wants and he/she can also add a tag on the tasks that he/she has performed.
- Use Bloc lib.
